How does it work?


Code-Trail helps you in making a flow chart of your code - use this for
planning code, investigating code, and sharing your insights of code.

  •  Each node on the graph is a line or a
     few lines of code
  •  Some lines will show you progress in
     the code
  •  Other lines show nodes that are
     inside content of the other nodes
  •  Perform Regex searches on your
     code and show results on graph
  •  Auto mapping of your git commits
  •  Save your charts
